What is CRUD?

CRUD is methods we use to interact with our databases utilizing queries
Check this link for more detailed information on CRUD.

  • Create

    • Create a entry in database by using the query:

    "INSERT INTO your_db_name
    (your_first_column_name, your_second_column_name, your_third_column_name)
    VALUES (first, second, third);"

  • Read

    • Read all entries in database by using the query:

    "SELECT * FROM your_db_name;"

  • Update

    • Update an entry in database by running the query:

    "UPDATE your_db_name
    SET your_first_column_name = value, your_second_column_name = value, your_third_column_name = value
    WHERE id = id # to update;"

  • Delete

    • Delete an entry in database by running the query:

    "DELETE FROM your_db_name
    WHERE id = id # to delete;"
